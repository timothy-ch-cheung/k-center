# Data set format
## Header
- first row has 6 values:
    - number of nodes [N]
    - number of centers [k]
    - number of blue nodes [B]
    = number of red nodes [R]
    - minimum blue coverage [b]
    - minimum red coverage [r]
- e.g. "[N] [k] [B] [R] [b] [r]"

# Subsequent data    
- subsequent N rows have 3 values:
    - x coordinate
    - y coordinate
    - colour class
